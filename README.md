# 杭州旅行攻略网站

一个美观的杭州三日游旅行攻略网页，包含：
- 天气信息展示
- 高德地图集成
- 美食推荐
- 住宿指南
- 交通出行
- 购物攻略

## 🚀 快速访问

由于Git环境配置问题，这里提供直接访问方式：

### 📂 本地访问
1. 下载所有文件到本地
2. 双击 `index.html` 文件，或用浏览器打开
3. 或使用Python启动本地服务器：
   ```bash
   python -m http.server 8080
   # 然后访问 http://localhost:8080
   ```

### 🌐 在线访问（推荐）
**最简单的方法**：使用GitHub网页界面直接部署

#### 步骤1：创建GitHub仓库
1. 访问 [github.com](https://github.com)
2. 点击右上角 "+" → "New repository"
3. 命名为 `hangzhou-travel-guide`
4. 选择 "Public"，勾选 "Add a README file"
5. 点击 "Create repository"

#### 步骤2：上传文件
1. 在新仓库页面，点击 "uploading an existing file"
2. 拖拽上传以下文件：
   - `index.html`（主页面）
   - `README.md`（项目说明）
3. 点击 "Commit changes"

#### 步骤3：启用GitHub Pages
1. 进入仓库的 "Settings" → "Pages"
2. Source选择 "Deploy from a branch"
3. Branch选择 "main" 和 "/ (root)"
4. 点击 "Save"

#### 步骤4：获得网址
等待几分钟后，您的网站将在：
`https://[您的用户名].github.io/hangzhou-travel-guide/`

## 🚀 部署平台对比

| 平台 | 优点 | 缺点 | 适合场景 |
|-----|------|------|----------|
| **GitHub Pages** | 免费、稳定、支持自定义域名 | 需要GitHub账号 | 长期使用、个人作品集 |
| **Netlify** | 拖拽部署、自动HTTPS、全球CDN | 免费版有流量限制 | 快速分享、测试部署 |
| **Vercel** | 现代化、性能优秀、自动部署 | 需要技术背景 | 专业项目、商业用途 |

## 💡 部署建议

**推荐顺序**：
1. **GitHub Pages** - 最稳定可靠，适合长期使用
2. **Netlify** - 如果想快速分享，可以同步部署
3. **Vercel** - 如果有技术背景，追求最佳性能
